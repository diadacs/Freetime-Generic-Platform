Folders

1Solution.2008 - contains solution files (sln) for Visual Studio 2008
1Solution.2010 - contains solution files (sln) for Visual Studio 2010

The solution files under the said folders are numbered in this format "00#.SolutionFileName.sln".

e.g. : 001.Freetime.Framework

To run the source build it from the lowest number to the highest.
